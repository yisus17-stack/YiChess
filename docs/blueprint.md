# **App Name**: YiChess Simplificado

## Core Features:

- Hero Inicio: Da la bienvenida a los usuarios y presenta el contenido del sitio web con un título principal, subtítulo y una imagen opcional relacionada con el ajedrez, guiando a los usuarios a explorar las reglas del ajedrez o las estrategias a través de un botón claro de llamada a la acción.
- Menú de Navegación: Proporciona una navegación fácil a través de las secciones del sitio web (Inicio, Reglas, Estrategias, Partidas, Contacto) a través de una barra de encabezado simple y persistente, resaltando la página activa para una orientación clara.
- Navegación Breadcrumb: Muestra la ubicación actual del usuario dentro de la estructura del sitio web, apareciendo justo debajo del menú principal para mejorar la orientación.
- Sección Reglas del Ajedrez: Explica las reglas del ajedrez y los movimientos de las piezas a través de texto claro y conciso, listas e imágenes posibles del tablero.
- Sección Estrategias y Tácticas: Ofrece consejos introductorios de ajedrez, cubriendo aperturas, jaque mates y planificación simple a través de texto organizado, listas o párrafos cortos, complementado con ejemplos de posiciones básicas.
- Sección Partidas Famosas: Destaca juegos de ajedrez históricos, dando descripciones concisas de partidas y movimientos cruciales, mejorado con imágenes de posiciones significativas del tablero.
- Formulario de Contacto: Permite a los usuarios enviar sugerencias o comentarios. Se realiza con validaciones HTML simples.
- Sistema de Búsqueda: Permite a los usuarios ingresar un término de búsqueda y enumera cualquier sección que contenga ese término de búsqueda como título.
- Página 404: Informa a los usuarios de una URL inexistente con un mensaje claro y un enlace directo de vuelta a la página de inicio, manteniendo un diseño simple y consistente.
- Pie de Página: Muestra información esencial de derechos de autor y políticas del sitio.

## Style Guidelines:

- Gris primario: #1a1a1a, para el texto y los elementos principales.
- Gris secundario: #4a4a4a, para texto y elementos de la interfaz de usuario menos prominentes.
- Color del borde: #e5e7eb, para dividir elementos y crear separación visual.
- Fondo neutro: #f9fafb, para un fondo claro y limpio.
- Fuente del cuerpo y del encabezado: 'Inter', una tipografía sans-serif, que proporciona una sensación limpia y moderna adecuada tanto para encabezados como para texto.
- Centrando el contenido para una experiencia de usuario limpia y enfocada.
- Estética minimalista con un fondo blanco o muy claro, asegurando la legibilidad.
- Sin animaciones, creando un entorno estático y libre de distracciones.
